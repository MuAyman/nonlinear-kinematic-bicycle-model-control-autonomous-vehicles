# Nonlinear Kinematic Bicycle Model Control

## Overview
This repository presents a **work-in-progress** implementation of a nonlinear kinematic bicycle model for ground vehicle motion and a comparative study of multiple trajectory tracking controllers. The project focuses on practical control design for vehicle dynamics, with emphasis on stability, tracking accuracy, and real-time feasibility.
